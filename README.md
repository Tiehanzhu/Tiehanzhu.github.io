# 朱轶豪的个人博客

> 西安电子科技大学 · 密码学硕士研究生

这是一个简约实用的个人博客，用于记录研究生学习与生活的点滴。

## ✨ 特点

- 🎯 **简约实用** - 专注内容，去除复杂功能
- 📝 **Markdown 写作** - 简单高效的写作体验
- ⏰ **时间线展示** - 按时间顺序展示所有文章
- 📱 **响应式设计** - 完美支持各种设备
- 🚀 **GitHub Pages** - 免费托管，自动部署

## 🚀 快速开始

### 本地运行

```bash
# 安装依赖
bundle install

# 启动服务
bundle exec jekyll serve

# 浏览器访问
http://localhost:4000
```

### 部署到 GitHub Pages

1. Fork 或克隆这个仓库
2. 将仓库重命名为 `你的用户名.github.io`
3. 修改 `_config.yml` 中的个人信息
4. 推送到 GitHub
5. 在仓库设置中启用 GitHub Pages
6. 访问 `https://你的用户名.github.io`

## 📝 写文章

在 `_posts` 文件夹中创建新文件：

```markdown
文件名：2025-11-20-my-post.md

内容：
---
layout: post
title: "文章标题"
date: 2025-11-20
tags: [标签1, 标签2]
---

## 正文

开始写你的内容...
```

详细说明请查看 `_posts/README.md` 或 `使用说明.md`。

## 📂 项目结构

```
├── _config.yml          # 网站配置
├── _layouts/            # 页面布局
├── _posts/              # 文章目录
├── assets/css/          # 样式文件
├── images/              # 图片资源
├── index.html           # 主页
├── archive.html         # 归档页面
├── about.html           # 关于页面
└── README.md            # 项目说明
```

## 🎨 自定义

### 修改个人信息

编辑 `_config.yml` 文件，更新你的姓名、学校、专业等信息。

### 修改样式

编辑 `assets/css/main.css` 文件，自定义颜色、字体、间距等。

### 修改页面

- `index.html` - 主页
- `archive.html` - 归档页面  
- `about.html` - 关于页面
- `_layouts/` - 布局模板

## 📄 页面说明

- **主页** - 个人介绍 + 最新文章列表
- **归档** - 按年份展示所有文章
- **关于** - 个人详细介绍

## 📖 技术栈

- [Jekyll](https://jekyllrb.com/) - 静态网站生成器
- [GitHub Pages](https://pages.github.com/) - 免费托管
- Markdown - 文章写作格式
- HTML/CSS - 页面和样式

## 📌 注意事项

1. 文章文件名必须是 `YYYY-MM-DD-title.md` 格式
2. 文件名中的标题部分使用英文
3. 文章头部信息格式要正确
4. 图片放在 `images/` 文件夹中
