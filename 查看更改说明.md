# 查看网站更改

## 问题：浏览器缓存

如果你在浏览器中没有看到变化，这是因为**浏览器缓存了旧的CSS文件**。

## 解决方法

### 方法1：强制刷新（推荐）

**Windows/Linux:**
- 按 `Ctrl + F5` 
- 或者 `Ctrl + Shift + R`

**Mac:**
- 按 `Cmd + Shift + R`

### 方法2：清除浏览器缓存

1. 打开浏览器开发者工具（F12）
2. 右键点击刷新按钮
3. 选择"清空缓存并硬性重新加载"

### 方法3：使用无痕模式

- 打开浏览器的无痕/隐私模式
- 访问 `localhost:4000`

## 如果还是看不到变化

运行以下命令重启Jekyll：

```bash
# 停止当前运行（Ctrl+C）
# 然后重新启动
bundle exec jekyll serve
```
