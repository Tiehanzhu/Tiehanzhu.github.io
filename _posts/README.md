# 文章写作指南

## 📝 文章命名规则

文件名格式：`YYYY-MM-DD-标题.md`

例如：`2025-11-20-my-first-post.md`

**注意**：文件名中的标题部分使用英文，避免使用中文。

## ✍️ 文章模板

### 基础模板

```markdown
---
layout: post
title: "文章标题"
date: 2025-11-20
tags: [标签1, 标签2]
---

## 开始

这里是文章内容...

## 正文

继续写你的内容...

## 结语

总结部分...
```

## 🏷️ 标签使用建议

常用标签：
- `论文阅读` - 论文笔记
- `学习方法` - 学习技巧
- `研究生生活` - 日常生活
- `目标` - 目标规划
- `总结` - 阶段总结
- `感悟` - 个人感悟
- `技术` - 技术相关
- `密码学` - 专业内容

根据实际情况自由添加标签，没有限制！

## 📖 Markdown 常用语法

### 标题
```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 文本样式
```markdown
**粗体文字**
*斜体文字*
~~删除线~~
```

### 列表
```markdown
无序列表：
- 列表项1
- 列表项2

有序列表：
1. 第一项
2. 第二项
```

### 链接和图片
```markdown
[链接文字](https://example.com)
![图片描述](/images/图片名.jpg)
```

### 代码
```markdown
行内代码：`code`

代码块：
\`\`\`python
def hello():
    print("Hello World")
\`\`\`
```

### 引用
```markdown
> 这是一段引用文字
> 可以多行
```

### 表格
```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 内容1 | 内容2 | 内容3 |
```

### 分割线
```markdown
---
```

## 🚀 快速创建新文章

### 方法一：直接创建文件

1. 在 `_posts` 文件夹中创建新文件
2. 文件名格式：`YYYY-MM-DD-title.md`
3. 复制上面的模板
4. 填写内容
5. 保存

### 方法二：复制示例文章

1. 复制一篇示例文章
2. 重命名为新的日期和标题
3. 修改头部信息（title, date, tags）
4. 替换正文内容

## 📌 重要提示

1. **日期格式必须正确**：`YYYY-MM-DD`（例如：2025-11-20）
2. **文件名使用英文**：避免使用中文字符
3. **头部信息格式**：
   - 开头和结尾都是 `---`
   - 字段名称后面是冒号和空格
   - 标签用方括号包裹，逗号分隔
4. **正文使用Markdown**：参考上面的语法说明
5. **图片存放**：将图片放在 `/images/` 文件夹中

## ✅ 文章头部信息检查清单

```yaml
---
layout: post          # 固定为 post
title: "标题"         # 必填，用引号包裹
date: 2025-11-20      # 必填，格式 YYYY-MM-DD
tags: [标签1, 标签2]  # 可选，方括号包裹
---
```

## 💡 写作建议

1. **定期更新**：建议每周至少写一篇
2. **保持真实**：记录真实的想法和感受
3. **结构清晰**：使用标题分段，便于阅读
4. **适当配图**：重要内容可以配图说明
5. **添加标签**：方便日后查找相关文章

## 📂 文章管理

- 所有文章都在 `_posts/` 目录中
- 按日期倒序显示（最新的在最前面）
- 可以通过标签筛选和查找
- 删除文章只需删除对应的 `.md` 文件
