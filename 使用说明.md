# 🎓 朱轶豪的个人博客 - 使用说明

## 📖 项目简介

这是一个简约大气的学术博客网站，基于 Jekyll + GitHub Pages 构建，专为记录研究生学习生活而设计。

### 设计理念
- ✨ **简约大气**：参考 Apple 和 Louis Vuitton 官网的设计风格
- 📝 **易于使用**：使用 Markdown 格式，写作简单高效
- 🎯 **分类清晰**：四大板块，内容井然有序
- 📱 **响应式设计**：完美支持手机、平板和电脑浏览

## 🚀 快速开始

### 方法一：直接部署到 GitHub Pages（推荐）

1. **创建 GitHub 仓库**
   - 仓库名必须是：`你的用户名.github.io`
   - 例如：`ZhwY1Ha0.github.io`

2. **上传代码**
   ```bash
   git add .
   git commit -m "Initial commit: 创建个人博客"
   git push origin main
   ```

3. **启用 GitHub Pages**
   - 进入仓库 Settings → Pages
   - Source 选择 `main` 分支
   - 等待 1-2 分钟

4. **访问网站**
   - 地址：`https://你的用户名.github.io`

### 方法二：本地预览（需要 Ruby 环境）

```bash
# 安装依赖
bundle install

# 启动服务
bundle exec jekyll serve

# 浏览器访问
http://localhost:4000
```

## ✍️ 如何写文章

### 第一步：创建文件

在 `_posts` 文件夹中创建新文件，命名格式：

```
YYYY-MM-DD-文章标题.md
```

例如：`2025-11-20-my-first-paper.md`

### 第二步：填写文章头部信息

```markdown
---
layout: post
title: "文章标题"
date: 2025-11-20
category: academic  # 选择：academic, life, goals, thoughts
tags: [标签1, 标签2]
---
```

### 第三步：编写正文内容

使用 Markdown 语法编写内容，参考 `_posts/README.md` 中的模板。

### 第四步：保存并推送

```bash
git add _posts/你的文章.md
git commit -m "新增文章：文章标题"
git push
```

## 🎨 个性化配置

### 修改个人信息

编辑 `_config.yml` 文件：

```yaml
# 作者信息
author:
  name: 朱轶豪
  bio: 西安电子科技大学密码学硕士研究生
  university: 西安电子科技大学
  major: 密码学
  email: 你的邮箱
  github: 你的GitHub用户名
```

### 添加头像

1. 准备一张正方形头像图片（建议 400x400px）
2. 命名为 `profile.png`
3. 放入 `images/` 文件夹
4. 在 `_config.yml` 中确认头像路径

### 修改网站标题

编辑 `_config.yml`：

```yaml
title: 你的名字
description: 你的简介
url: https://你的用户名.github.io
```

## 🔧 常见问题

### Q: 文章写完后多久能看到？
A: GitHub Pages 通常 1-2 分钟会自动构建，本地运行则是实时的。

### Q: 如何删除示例文章？
A: 直接删除 `_posts/` 中的示例文件即可。

### Q: 如何修改样式？
A: 编辑 `assets/css/main.css` 文件。

### Q: 图片放在哪里？
A: 放在 `images/` 文件夹，文章中使用：
```markdown
![描述](/images/图片名.jpg)
```

1. **文件名必须是英文**：`_posts` 中的文件名不要使用中文
2. **日期格式要正确**：`YYYY-MM-DD`
3. **提交前检查**：确保 YAML 头部格式正确
4. **定期备份**：虽然 Git 有版本控制，但重要内容建议额外备份
